# 第3回
## 迷路ゲーム：迷えるこうかとん(ex03/maze.py)
### ゲーム概要
- ex03/maze.pyを実行すると，1500x900のcanvasに迷路が描画され，迷路に沿ってこ
うかとんを移動させるゲーム
- 実行するたびに迷路の構造は変化する

### 操作方法
- 矢印キーでこうかとんを上下左右に移動する

### 追加機能
- ゴールオブジェクトの追加
- 敵オブジェクトの追加

### 実装仕切れなかった機能
- ゴール、敵オブジェクトのランダム配置
- 敵オブジェクトの行動パターン追加

### メモ
- 敵オブジェクトは自身が一マス動くと一マス動く
- 迷路を広くしないと敵オブジェクトの動き次第で難易度が高くなる可能性がある
